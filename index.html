salut tous le monde, je voulais vous proposer une petite liste des commandes les plus importantes à maitriser sur le terminal git afin de pouvoir faire des repository sur github.

    Le terminal ( le git)

      pwd+entrée= connaitre le répertoire courant
      Ls +entrée=liste des fichiers et repertoire dans le dossier courant
      cd+ du nom du fichier/repertoires= permet de se déplacer dans les repertoires
      touch+nom fichier=permet de creer un fichier directement du terminal
      mkdir+nom/dossier= permet de creer des dossiers/repertoires
      cat=afficher le contenu des dossiers

    configurer un git

      git config --global user.name"votre pseudo"=configurer son pseudo
      git config --global user.email"votre email"=configurer son email

    faire un commit

    Avant de faire un commit, il faut creer un dossier et des fichier grace au commande(touch)

      git add +fichier= rajouter un fichier
      git commit -m,-a ou -m-a= enregistrer dans le reseau local
      git log(qpour quitter le log)= vous permet de lire l'historique des derniers commit(modifications apportées)
